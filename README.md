# 🎓 EduTrack - Student Management System

A comprehensive student management system for tracking grades, attendance, and generating performance reports.

## ✨ Features
- ✅ Add, edit, delete student records
- 📊 Track grades across multiple subjects
- 📅 Attendance tracking with percentages
- 📈 Performance analytics with charts
- 🔍 Search and filter students
- 📋 Generate detailed reports
- 💾 Export/import data (JSON)
- 📱 Fully responsive design
- 🌓 Clean, modern interface

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript (ES6+)
- Chart.js for data visualization
- Font Awesome Icons
- Local Storage API

live demo https://sandilenkosi12.github.io/student-management-system/

## 📦 Installation
1. Clone the repository
2. Open `index.html` in your browser
3. Start managing students!

## 📝 Usage
- Click "Add New Student" to add records
- Use the search bar to find students
- Click edit/delete buttons to modify records
- View analytics in the charts section
- Export data for backup
- Generate reports for printing
